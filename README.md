## Recommended Use Cases

- **Synthetic data generation** — detection, segmentation, and tracking pipelines
- **Domain randomization** — robot perception training with varied scene configurations
- **Manipulation benchmarks** — kitchen and tabletop object interaction tasks
- **Multimodal reasoning** — tasks grounded in 3D scenes combining vision and language
